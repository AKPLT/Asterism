# Asterism — 社内ツールポータル

## 課題

社内には開発・業務効率化のためのツールが多数存在するが、現状は以下の問題を抱えている。

- **散在している** — 共有フォルダ・Teams・個人PCなど置き場がバラバラで、どこに何があるか分からない
- **バージョン管理がない** — 誰がどのバージョンを使っているか把握できず、古いバージョンが混在する
- **更新が届かない** — ツールを更新しても全員に行き渡るまでに時間がかかる・届かない人が出る
- **インストールが手間** — ZIP展開・設定ファイルの配置など、IT知識がないと詰まりやすい

---

## 解決策：Asterism

**社内ツールのダウンロード・更新・起動を一元管理するポータルアプリ**

利用者はポータルを開くだけで、利用可能なすべての社内ツールを確認・インストール・起動できる。  
管理者はポータル上でツールを登録・更新するだけで、全利用者に最新版が届く。

---

## 主な機能

### 利用者向け

| 機能                     | 説明                                                                             |
| ------------------------ | -------------------------------------------------------------------------------- |
| ツール一覧               | カード形式で一覧表示。カテゴリ・タグ・キーワードで絞り込める                     |
| ワンクリックインストール | 「インストール」ボタンを押すだけでZIPをダウンロード・展開                        |
| 自動更新通知             | バックグラウンドで定期確認（10分ごと）。更新があるとバナーとトレイ通知でお知らせ |
| ワンクリック起動         | インストール後は「起動」ボタンでそのままツールを開ける                           |
| お気に入り               | よく使うツールをお気に入り登録してすぐアクセス                                   |
| オフライン対応           | サーバーに繋がらなくても、最後に取得した一覧をキャッシュから表示                 |
| システムトレイ常駐       | ウィンドウを閉じてもバックグラウンドで更新監視を継続                             |

### 管理者向け（パスワードでロック解除）

| 機能             | 説明                                                                             |
| ---------------- | -------------------------------------------------------------------------------- |
| ツール登録       | GUIからZIP/アイコンをアップロードするだけで登録完了                              |
| ツール編集・更新 | 新しいZIPをアップロードしてバージョンを上げると、全利用者に更新通知が届く        |
| ツール無効化     | 一時的に非表示にしたいときに使用。削除はしないので復元も簡単                     |
| ツール削除       | 一覧から除去。ファイルも一緒に削除するか選択できる（誤操作防止の二段階確認あり） |

## システム構成

```
社内ネットワーク

  [サーバーPC × 1台]
  ┌──────────────────────────────┐
  │  Asterism.Server.exe         │
  │  ASP.NET Core (.NET 8)       │
  │                              │
  │  ・manifest.json の管理       │
  │  ・ツールZIP・アイコンの保管  │
  │  ・管理API（認証付き）        │
  └──────────────────────────────┘
              ↑↓ HTTP（社内LAN）
  [クライアントPC × 全利用者]
  ┌──────────────────────────────┐
  │  Asterism.Client.exe         │
  │  WPF (.NET 8) Windows専用    │
  │                              │
  │  ・ツール一覧・検索・絞り込み │
  │  ・インストール・起動・更新   │
  │  ・管理者モード（同梱）       │
  └──────────────────────────────┘
```

**必要なサーバーは1台だけ。** 既存の社内PCでも動作可能。  
クライアントは`.exe`を配布するだけで動作（.NETランタイム不要）。

## 利用フロー（一般ユーザー）

```
① Asterism を起動
       ↓
② ツール一覧からインストールしたいツールを探す
   （カテゴリ絞り込み・キーワード検索）
       ↓
③「インストール」ボタンをクリック
   → ZIP が自動ダウンロード・展開される
       ↓
④「起動」ボタンをクリックでツールが開く

── バックグラウンドで定期確認 ──

⑤ 管理者がサーバー側でバージョンアップ
       ↓
⑥「更新あり」バッジとバナー通知が届く
       ↓
⑦「更新」ボタンをクリックで最新版に切り替わる
```

---

## 利用フロー（管理者）

```
① メニュー「ツール → 管理者モード」でパスワード入力
       ↓
② 管理画面が開く
       ↓
③「新規登録」→ ZIPとアイコンを選択して保存
   → サーバーに自動アップロード・manifest.json 更新
       ↓
④ 全利用者の Asterism に新ツールが反映される
   （次の更新確認タイミング、最長10分後）
```

## 技術スタック

| 項目         | 採用技術                                                        |
| ------------ | --------------------------------------------------------------- |
| クライアント | C# / WPF / .NET 8（Windows専用）                                |
| サーバー     | C# / ASP.NET Core Minimal API / .NET 8                          |
| 認証         | APIキー方式（Bearerトークン）。パスワードはサーバー側のみで管理 |
| データ形式   | JSON（manifest.json）+ ファイル配信                             |
| 追加インフラ | **不要**（DBなし・クラウドなし・追加ミドルウェアなし）          |

---

## 今後の拡張候補

- **クライアント自己更新** — AsterismをAsterism自身で配布する（仕組みは既に対応済み）
- **HTTPS対応** — 社外アクセスが必要な場合はリバースプロキシでHTTPS化
- **ユーザー認証** — 部門ごとに見えるツールを制限したい場合
- **macOS/Linux対応** — クライアントをMAUI/Avalonia等に移植する場合
- **インストーラー型ツール対応** — `.exe`/`.msi` 形式のインストーラーも配布可能（実装済み）

## まとめ

| Before                                  | After                              |
| --------------------------------------- | ---------------------------------- |
| ツールがどこにあるか分からない          | ポータルを開けば全ツールが一覧表示 |
